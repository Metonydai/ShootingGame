# Shooting Game

A mini third-person shooting game made with Unity 6, following the YouTube series [Create a Game](https://www.youtube.com/playlist?list=PLFt_AvWsXl0ctd4dgE1F8g3uec4zKNRV0) by Sebastian Lague. 

### Features
- Player controller
- Weapon system
- Enemy spawner
- Enemy AI
- Map generation
- Visual effects
- Audio
- UI
